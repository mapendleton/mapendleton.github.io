Kafka API

Created an API for creating/altering/deleting topics, and Partners (basically users), and for granting/removing access to said topics using the confluent MDS API. This replaced the manual process for teams to request and create topics with an dev friendly api.

Kafka on K8s POC

Created a POC using two apps to demonstrate deploying kafka on kubernetes, a producer and consumer. The Producer was written in Java using the Springboot framework and exposed some endpoints for producing to a topic. The Consumer was a React/Node app that used the exposed endpoints to produce messages to an app and then listened to that same topic using a websocket so as to display the messages being sent to the topic in real time.

Message Validator

Created a Message Validator for a Kafka Producer, it would read in validation rules written in yaml and validate avro messages to make sure they followed the custom rules.

Reassign Partitions

Created a python script which would log in to a kafka broker check the disc sizes and balance them if they weren't balanced. So if there were 4 data discs, and one was 90% full while the others were 10% full then it would balance them by moving partitions until the 4 discs were close in size.
